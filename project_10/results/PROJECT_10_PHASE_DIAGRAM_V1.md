# PROJECT 10 PHASE DIAGRAM V1
## First Conceptual Phase Diagram for Rescue Behavior

**Date:** April 2026  
**Project:** 10  
**Status:** ACTIVE PHASE-DIAGRAM NOTE

---

## 1. Purpose

This document defines the first conceptual phase diagram for rescue behavior in Project 10.

The purpose is not to claim a final physical-style phase theory.

It is to state clearly that the current evidence is now best understood through a structured regime map with transition regions.

This is the first phase-diagram-style representation in the project.

---

## 2. Phase Axes

The current phase diagram is defined by two main axes:

### Axis A — Family-Level Heterogeneity
Increasing from:
- low
to
- medium
to
- high

### Axis B — Universal Rescue Power
Increasing from:
- baseline
to
- moderate
to
- strong

These two axes currently define the clearest tested rescue regime space.

---

## 3. Current Regions

The current evidence suggests at least three qualitative regions.

### Region I — Family-Aware Advantage Region
This region appears when:
- heterogeneity is sufficiently strong
- and universal rescue power remains baseline

Here:
- family-aware rescue outperforms universal rescue

---

### Region II — Boundary / Transition Region
This region appears near:
- low-to-medium heterogeneity transition at baseline power
- and baseline-to-moderate power transition at medium/high heterogeneity

Here:
- outcomes become near-tied or mixed
- neither rescue logic dominates strongly

---

### Region III — Universal Dominance Region
This region appears when:
- universal rescue power becomes moderate or strong

Here:
- universal rescue overtakes family-aware rescue across the tested map

---

## 4. Current Best Diagram Reading

The strongest current reading is:

- rescue behavior changes qualitatively across regime-space regions
- the shifts are not random
- and the current evidence supports transition-like behavior between regions

This is why a phase-diagram-style representation is now useful.

---

## 5. Why This Matters

This matters because it changes the theoretical style of Project 10.

The project is no longer best described only by:
- laws

It is now also describing:
- regions

This is a substantial theoretical advance.
