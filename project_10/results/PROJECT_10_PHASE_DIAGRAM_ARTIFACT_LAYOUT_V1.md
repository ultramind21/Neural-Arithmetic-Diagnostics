# PROJECT 10 PHASE DIAGRAM ARTIFACT LAYOUT V1
## Layout Structure for the First Rescue Phase Diagram

**Date:** April 2026  
**Project:** 10  
**Status:** ACTIVE ARTIFACT LAYOUT

---

## 1. Purpose

This document defines the layout structure for the first rescue phase diagram artifact in Project 10.

The goal is to specify how the conceptual content should be spatially organized before any rendering is attempted.

---

## 2. Overall Layout

The artifact should use a single main panel.

That panel should contain:

- horizontal axis for universal rescue power
- vertical axis for family-level heterogeneity
- three main qualitative regions
- visible transition bands
- labels showing boundary drift

The layout should remain simple and theory-facing.

---

## 3. Region Placement

### Lower-left area
This area should represent:

**Transition / Mixed**
or weak family-aware separation at low heterogeneity and baseline power.

### Central-left to upper-left area
This area should represent:

**Family-Aware Advantage**

This region should occupy the baseline-power side at sufficiently strong heterogeneity.

### Right-hand side
This area should represent:

**Universal Dominance**

This region should expand as universal rescue power increases.

---

## 4. Transition Bands

The layout should display:

### A. Heterogeneity transition band
A narrow vertical local band near baseline power,
marking the shift from mixed behavior to family-aware advantage.

### B. Power transition band
A slanted or shifted band separating:
- family-aware advantage / mixed behavior
from
- universal dominance

This band should visibly drift upward with heterogeneity.

---

## 5. Boundary Drift Representation

The artifact should make the following visible:

- the universal-dominance boundary occurs at lower power for medium heterogeneity
- and at higher power for high heterogeneity

This is the key geometric feature.

It should not be hidden.

---

## 6. Label Placement

### Title
Top of figure

### Subtitle
Directly below title

### Axis labels
Standard placement:
- horizontal label below panel
- vertical label left of panel

### Region labels
Inside the regions:
- Family-Aware Advantage
- Transition / Mixed
- Universal Dominance

### Boundary drift label
Placed near the shifting boundary line/band

---

## 7. Caption Placement

A short caption should appear below the figure panel.

It should summarize:
- threshold-structured rescue behavior
- region dependence
- and heterogeneity-dependent boundary drift
