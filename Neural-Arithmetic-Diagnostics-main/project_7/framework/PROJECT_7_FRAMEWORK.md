# PROJECT 7 FRAMEWORK
## From Local Mechanistic Competence to Global Compositional Failure

**Date:** April 2026  
**Status:** ACTIVE FRAMEWORK  
**Relation to previous work:** New project after Projects 5 and 6

---

## 1. Core Question

Project 7 asks:

> Why can a model show meaningful local arithmetic competence and interpretable internal structure, yet still fail at global compositional behavior?

This is the central question of the project.

---

## 2. Why This Project Exists

Project 5 established:
- decomposition can work structurally
- learned local decomposition fails in a structured way
- some representational interventions help only partially

Project 6 established:
- arithmetic-relevant internal structure exists
- some units are causally important
- internal specialization exists
- sequence-level trajectory patterns matter
- but simple mechanistic explanations remain insufficient

Together, these projects reveal a deeper problem:

- local competence exists
- local structure exists
- yet global family-level or compositional failure remains

Project 7 is designed to study that gap directly.

---

## 3. Core Hypothesis

The central working hypothesis is:

> The main unresolved bottleneck lies in the transition from local mechanistic competence to stable global compositional behavior.

This may reflect:
- error propagation
- interface mismatch
- unstable composition
- or a deeper local-to-global representational gap

Project 7 exists to test these possibilities.

---

## 4. Main Research Goal

The goal is not to find another isolated probe.

The goal is to identify:

- what local competence is insufficient for
- where the local-to-global bridge breaks
- and what kind of mechanism is required for composition-level robustness

---

## 5. Main Project Strategy

Project 7 should explicitly combine:

### A. Local competence evidence
from Project 5 and Project 6

### B. Composition failure evidence
from decomposition and family-level behavior

### C. Bridge experiments
that test how local structure is or is not transferred into global success

---

## 6. First Planned Direction

The first Project 7 experiments should focus on:

> **local-to-global bridge diagnostics**

This includes:
- composition-level tracing
- interface-state propagation
- stepwise error amplification
- and transition of local hidden structure into sequence-level outcomes

---

## 7. Non-Goals

Project 7 does not start by:
- searching for a new state-of-the-art model
- rewriting all previous projects
- or claiming a final theory from the first experiment

It starts from:
- a sharp question
- a rich previous foundation
- and the need to explain a specific unresolved gap
