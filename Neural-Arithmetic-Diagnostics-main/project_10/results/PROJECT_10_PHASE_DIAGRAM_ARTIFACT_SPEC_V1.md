## 1. Purpose

This document defines the first official artifact specification for the Phase 2 rescue phase diagram in Project 10.

The purpose is to create a stable conceptual representation of the current theory state.

This artifact is not yet a final publication figure.
It is the first formal diagram specification.

---

## 2. Core Content the Artifact Must Show

The artifact must show:

1. the regime-space axes
2. the main qualitative regions
3. the transition bands
4. the current local boundary estimates
5. the fact that the power boundary shifts with heterogeneity

These are the non-negotiable contents.

---

## 3. Required Axes

### Horizontal axis
**Universal Rescue Power**

### Vertical axis
**Family-Level Heterogeneity**

This orientation should be used consistently unless a later revision explicitly changes it.

## 4. Required Regions

The artifact must include at least three qualitative regions:

### Region I
**Family-Aware Advantage Region**

### Region II
**Transition / Mixed Region**

### Region III
**Universal Dominance Region**

These regions must be visually distinct.
