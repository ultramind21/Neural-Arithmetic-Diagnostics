## Purpose

This document records the first perturbation-map result in Project 9.

Its purpose was to test whether the first higher-dimensional arithmetic-like sandbox exhibits meaningful local-to-global sensitivity, and whether that sensitivity is uniform or position-dependent.

---

## Main Result

The first 3D lattice produced a clear local-to-global effect:
- perturbing one local cell changed multiple other cells in the lattice

At the same time, the result showed something subtler:
- all tested positions produced the same number of changed cells
- but they did not produce the same carry-dynamic outcome

This is the first topology-sensitive result in Project 9.

---

## What This Establishes

This result establishes the following:

1. the first higher-dimensional sandbox is nontrivial
2. local perturbation can propagate globally
3. local-to-global reach exists in this space
4. simple influence-count symmetry does not imply identical arithmetic or carry response

This is already enough to justify the sandbox experimentally.

---

## Symmetry and Asymmetry

### Structural symmetry
In the current `2×2×2` lattice:
- each perturbed position changes the same number of cells

This means the first sandbox is symmetric with respect to perturbation reach count.

### Arithmetic asymmetry
Despite that structural symmetry:
- some perturbations preserve total carry count
- some decrease it
- some increase it

This shows that the sandbox is already richer than a trivial symmetric toy.

---

## Correct Interpretation

The strongest safe interpretation is:

> the first 3D arithmetic-like sandbox is structurally symmetric in perturbation reach at the level tested, but already non-uniform in carry-dynamic response.

This is a useful and scientifically meaningful first result.

---

## Why This Matters

This matters because it shows that Project 9 is not merely a conceptual extension.

The sandbox already has:
- measurable propagation
- nontrivial arithmetic response variation
- and a meaningful difference between topology-level and carry-level behavior

That is enough to justify further development.

---

## What This Does NOT Yet Prove

This result does not yet prove:

- that the 2×2×2 lattice is the best sandbox size
- that higher-dimensional composition behaves fundamentally differently from linear composition in every respect
- that the perturbation map is fully understood
- or that family-like behavior has already emerged in the 3D space

It proves only that:
> the first 3D sandbox is scientifically alive and already exhibits meaningful local-to-global arithmetic structure.

---

## Recommended Next Step

A natural next step is now:

- compare multiple perturbation magnitudes
- compare larger lattice shapes
- test whether larger lattices break the current reach symmetry
- and begin defining family-like structure inside the higher-dimensional world
