# PROJECT 9 FRAMEWORK
## High-Dimensional Compositional Sandboxes

**Date:** April 2026  
**Status:** ACTIVE FRAMEWORK  
**Relation to previous work:** New project after Projects 4–8

---

## 1. Core Question

Project 9 asks:

> How does local competence behave when composition takes place in a higher-dimensional structured state space rather than a single linear sequence?

This is the central question of the project.

---

## 2. Why This Project Exists

Projects 4–8 established:

- high arithmetic accuracy is not enough
- local competence does not automatically become global robustness
- family-level failures are heterogeneous
- internal arithmetic structure is real
- architecture design can rescue different failure families through different mechanisms

But all of those results still emerged mainly in relatively linear compositional settings.

Project 9 exists to test whether those findings:
- persist
- change
- or deepen

when composition is embedded in a higher-dimensional structured state space.

---

## 3. Main Project Direction

Project 9 should be understood as a **sandbox expansion project**.

It is not only asking:
- what model performs best

It is asking:
- how compositional structure itself changes when the space of composition becomes richer

This makes it the first project in the line that shifts from:
- arithmetic sequence reasoning
to
- structured high-dimensional compositional worlds

---

## 4. Phase 1 Focus

Project 9 will begin with one bounded but powerful branch:

# **3D Compositional Arithmetic Sandbox**

This branch should:
- preserve arithmetic-like local rules
- allow local-to-global cascades
- introduce higher-dimensional structural interaction
- remain controlled enough for diagnosis and intervention

---

## 5. Why 3D First

A 3D sandbox is the most natural first leap because it keeps continuity with the current research line while expanding the compositional space substantially.

It allows the project to test:
- whether current local-to-global failure logic is specific to linear sequences
- whether interfaces and controllers behave differently in richer topology
- whether failure and rescue mechanisms become more complex in higher-dimensional composition

---

## 6. Main Scientific Goal

The main scientific goal is:

> determine whether the local competence vs global robustness gap discovered in earlier projects is a specifically linear-sequence problem or a more general compositional phenomenon.

This is the deepest reason Project 9 matters.

---

## 7. Initial Research Layers

Project 9 should eventually support at least four layers:

### Layer A — Sandbox Design
What is the structure of the 3D arithmetic world?

### Layer B — Local Rule Definition
What local arithmetic or state-update rules govern the sandbox?

### Layer C — Failure / Rescue Analysis
How do local competence, family structure, and architecture support behave in this richer setting?

### Layer D — Mechanistic Expansion
Do Projects 6–8 style interpretability and architecture ideas transfer into the higher-dimensional world?

---

## 8. Non-Goals

Project 9 does not begin by:
- trying to solve every high-dimensional combinatorial problem
- becoming a generic cube-puzzle simulator
- or replacing all previous projects

It begins with:
- one careful sandbox
- one structured jump beyond linear composition
- and one new class of compositional question
