# Project 11 — Executive Abstract (April 2026)

Project 11 tests whether a regime-theory style predictor (compact, interpretable rules) can reliably classify outcomes in a structured synthetic rescue system, and how this compares to local interpolation baselines (nearest neighbor). The work first validates the metric knobs (H, P) in a minimal foundation system, then introduces strict prediction gates and transfer tests across multiple system variants. A key discovery is that **hard clamping** (piecewise saturation) induces boundary instability and concentrates rule errors; replacing hard clamp with **soft saturation** (soft-clamp, k=15) restores smoother structure where an interpretable rule baseline (V3.1) becomes competitive.

Under soft labels, dense nearest-neighbor improves with resolution (e.g., NN81 macroF1_present=0.9847) but at increased reference cost. Crucially, Project 11 shows **structure-guided sampling**: combining uniform coverage with boundary-focused sampling (using rule-derived boundary score for selection) achieves near-dense performance with far fewer points (e.g., N=1000 mixed reaches macroF1_present≈0.9780 mean over seeds). Boundary-only sampling fails, demonstrating that global coverage is necessary. The packaged outputs provide an evidence matrix, key claims, and figure-ready tables for publication.
