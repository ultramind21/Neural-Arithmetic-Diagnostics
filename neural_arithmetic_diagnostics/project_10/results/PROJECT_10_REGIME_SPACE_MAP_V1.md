## 1. Purpose

This document records the first explicit regime-space map produced in Project 10.

It summarizes the Phase 2 threshold matrix in a compact theory-facing form.

The purpose is to make clear that rescue behavior is best understood as position-dependent within a structured regime space.

---

## 2. Regime Axes

The current regime map is defined by two axes:

### Axis A — Family-Level Heterogeneity
- low
- medium
- high

### Axis B — Universal Rescue Power
- baseline
- moderate
- strong

These axes define the first tested rescue regime space.

---

## 3. Current Regime Map

| Heterogeneity \ Universal Power | Baseline | Moderate | Strong |
|---------------------------------|----------|----------|--------|
| **Low**    | boundary / near-tie | universal dominance | universal dominance |
| **Medium** | family-aware advantage | universal dominance | universal dominance |
| **High**   | family-aware advantage | universal dominance | universal dominance |

This is the central result of the current map.

---

## 4. Main Interpretation

The current map shows that:

- family-aware rescue advantage is not globally stable
- universal rescue dominance is not globally guaranteed either
- the outcome depends on regime position

More specifically:

- low heterogeneity at baseline produces a boundary case
- medium/high heterogeneity at baseline produce family-aware advantage
- any tested increase in universal power shifts the system toward universal dominance

This is the strongest current map-level finding.

---

## 5. Why This Matters

This map is important because it replaces an overly simple higher-order law picture with a stronger, more precise theory structure.

Instead of asking:
- "which rescue is always better?"

Project 10 can now ask:
- "in which regimes does each rescue logic win?"

That is a better scientific question.

---

## 6. What This Changes

This map changes the theoretical framing of Project 10 in an important way:

- it weakens broad necessity claims
- it strengthens bounded comparative claims
- and it suggests that rescue theory may be phase-like or threshold-like rather than law-like in a simple universal sense

This is a major theoretical shift.

---

## 7. Current Best Working Principle

The strongest current higher-level principle is:

> rescue behavior is regime-dependent, and the comparative advantage of family-aware versus universal rescue is jointly controlled by family-level heterogeneity and universal rescue power.

This is the current best top-level statement supported by the map.
