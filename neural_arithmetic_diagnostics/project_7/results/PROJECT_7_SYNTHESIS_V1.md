# PROJECT 7 SYNTHESIS V1
## From Local Competence to Differential Failure Mechanisms

**Date:** April 2026  
**Project:** 7  
**Status:** ACTIVE SYNTHESIS  
**Scope:** first synthesis, not final closure

---

## 1. Purpose

This document records the first synthesis point of Project 7.

Project 7 was created to answer a deeper question emerging from Projects 5 and 6:

> Why can local arithmetic competence and meaningful internal structure coexist with family-level compositional failure?

The experiments completed so far now support a much sharper answer than the project had at entry.

---

## 2. Starting Position of Project 7

Project 5 established:
- decomposition can work structurally
- learned local processing can still fail compositionally
- some fixes help only partially
- and family-level failure is more structured than simple decomposition breakdown

Project 6 established:
- arithmetic-relevant internal structure exists
- local units can be carry-sensitive
- successful and failing local cases are internally distinguishable
- some units matter causally
- and local internal structure is not enough for a simple direct family-level explanation

Project 7 began where those two projects met:
- the gap between local competence and global failure

---

## 3. First Project 7 Result: Stepwise Composition Trace

The first Project 7 result showed that family-level failures are not well described as uniform collapse.

Instead:
- some failing families contain recurring local failure points
- those failures are localized and structured
- later positions may remain locally correct even after earlier errors

This was the first direct bridge result from local behavior to sequence-level failure.

The strongest consequence of that result was:
> the local-to-global gap is structured, not diffuse.

---

## 4. Second Project 7 Result: Context-Trigger Intervention

The second Project 7 result then tested whether the identified local trigger contexts were causally sufficient to explain family-level failure.

The outcome was a breakthrough:

- one family (`block_boundary_stress`) was rescued completely by the trigger correction
- another family (`alternating_carry`) was not rescued by the same intervention
- the successful control family (`full_propagation_chain`) remained unaffected

This means the family-level failures are not just structured.
They are also:
# **mechanistically different**

---

## 5. What Project 7 Has Now Established

Project 7 now establishes the following:

### A. Family-level failure is not monolithic
Different families do not fail for one single reason.

### B. Some failures are trigger-correctable
At least one family can be rescued through a sharply targeted local intervention.

### C. Some failures are trigger-insensitive
At least one other family remains failed under the same local correction.

### D. Local-to-global breakdown is now partially decomposed
The problem is no longer just:
- "Why does composition fail?"

It is now:
- "Which failure mode is active in which family, and which interventions affect which family?"

This is a major increase in mechanistic resolution.

---

## 6. Why This Changes the Problem

Before Project 7, a natural but overly simple view was:

- local arithmetic fails to compose globally

Project 7 now replaces that with a more precise picture:

- local competence can be strong
- local failure can be recurring and structured
- some local triggers are sufficient to explain one family
- but not another

This means that the local-to-global bridge problem is not one problem.
It is a family of related but distinct mechanistic problems.

That is the deepest current result of the project.

---

## 7. Strongest Current Scientific Position

The strongest current Project 7 position is:

> local arithmetic competence fails to scale globally through multiple distinct family-level mechanisms, not through one uniform compositional breakdown.

This is the most important scientific statement produced by Project 7 so far.

It is stronger than saying:
- "some families fail"
because it explains that
- **different families fail for different local-to-global reasons**

---

## 8. Why This Matters for the Full Research Line

This result matters beyond Project 7 itself.

Projects 4–6 showed:
- that behavior can be narrow
- that internal structure can be real
- and that local causal structure can be identified

Project 7 now adds:
- local-to-global bridge structure is itself heterogeneous

This means the research line has moved from:
- observing narrow performance
to
- diagnosing why different forms of narrowness arise

That is a major scientific advance.

---

## 9. What Project 7 Does NOT Yet Establish

Project 7 does not yet establish:

- a complete taxonomy of all failure mechanisms
- a final mechanistic explanation of `alternating_carry`
- a full causal decomposition of all families
- a final solution to local-to-global arithmetic composition
- closure of the global compositional reasoning problem

Those remain open.

---

## 10. Why This Is Already a Strong Result

Even without final closure, Project 7 has already produced something very strong:

- a direct local-to-global bridge experiment
- a successful trigger-based rescue
- and a demonstration that another family resists the same fix

This is not a minor descriptive insight.
It is a real mechanistic differentiation result.

---

## 11. Current Best Interpretation

The most defensible current interpretation is:

> global compositional failure in this arithmetic setting is not governed by one shared mechanism. At least one family is strongly trigger-driven, while another is not rescued by the same local intervention. This means that local-to-global failure must be studied as a multi-mechanism problem.

This is the strongest synthesis statement currently supported by Project 7.

---

## 12. Recommended Next Step

The next meaningful question is now:

> What specifically distinguishes the trigger-insensitive failure family from the trigger-correctable one?

That is the strongest next mechanistic frontier available.

It suggests future work focused on:
- trigger-insensitive failure structure
- family-specific bridge mechanisms
- and richer causal decomposition of local-to-global failure modes
